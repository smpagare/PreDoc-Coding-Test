# CenTax Research Economist Coding Test

**Author:** Siddhant Pagare

## Overview

This replication package contains the R code and data for the CenTax Research Economist coding test. The analysis investigates the effect of a tax incentive policy (introduced in the 2007-2008 UK tax year) on charitable giving via Gift Aid, using a regression discontinuity design (RDD) at the £34,600 earned income threshold.

## Folder Structure

```
CenTax Research Economist Coding Test/
├── README.md              <- This file
├── data_analysis.r        <- Main R script (all 10 tasks)
├── data/
│   ├── ind_0607.csv       <- Individual-level data, baseline year (2006-2007)
│   ├── ind_0708.csv       <- Individual-level data, treatment year (2007-2008)
│   └── hh_0708.csv        <- Household-level data (2007-2008)
└── output/                <- Generated by the script
    ├── analysis_cleaned.csv
    ├── T3_giftaid_vs_income.png
    └── T4_giftaid_trimmed_vs_income.png
```

## How to Replicate

1. Open `data_analysis.r` in R or RStudio.
2. Update the `setwd()` path on line 6 to point to this folder on your machine.
3. Run the entire script. All outputs will be saved to the `output/` folder.

## Requirements

- R (tested on version 4.5.1)
- R packages: `tidyverse`, `ggplot2`
